    • Challenge Project: Build a Website Design System

    Build a style guide/design system for your website.

    This should include sections on :

    • Colors
      • Each color should have a name, a box displaying the color, the CSS value used to create the color in hex or rgb and an optional description of how you intend to use the colors.
    • Fonts
      • Each font should include, the font name, a series of styles you intend to use (e.g. bold, italic), and each style should include a series of words to demonstrate the text style. Make sure to display each of the sections in the proper font, size, and style.
    • Specific Text Styles
      • Include at least 3 elements and their style rules. (e.g. Subheading/h2: font-weight, size, font-family)
